## install ##

From rubygems:  
~~~
    [sudo] gem install num4tststatistic2
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4tststatistic2.git  
    cd num4tststatistic2  
    gem build *.gemspec
    gem install *.gem
~~~
